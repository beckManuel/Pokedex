# Pokédex

## Overview
Built on android studio using Kotlin in 5 days as a code challenge. Small Pokédex App that allow a user to search information about a specific Pokémon!


### Components used
* MVVM
* Navigation Components
* Data Binding
* Dependency Injection
* Kotlin Coroutines
